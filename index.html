Guia Rápido

Windows

Baixe o arquivo e execute. Se o Windows SmartScreen aparecer, clique em "Mais informações" e "Executar assim mesmo".

Linux

Dê permissão de execução ao arquivo AppImage:

chmod +x RocknixManager.AppImage

macOS

Após mover para a pasta Aplicativos, você pode encontrar um aviso de segurança. Para abrir:

  1. Clique com o botão direito no ícone do Rocknix Manager.
  2. Selecione "Abrir" no menu.
  3. Na janela, clique em "Abrir" novamente.
