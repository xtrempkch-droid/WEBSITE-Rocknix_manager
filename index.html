Guia de Instalação

Windows

Basta baixar o arquivo e executar. Se o Windows SmartScreen aparecer, clique em "Mais informações" e "Executar assim mesmo".

Linux

Dê permissão de execução ao arquivo baixado:

chmod +x RocknixManager.AppImage

macOS

Após baixar e mover para a pasta Aplicativos, você pode encontrar um aviso de segurança. Para abrir:

  1. Clique com o botão direito (ou Control + Clique) no ícone do Rocknix Manager.
  2. Selecione "Abrir" no menu.
  3. Na janela que aparecer, clique em "Abrir" novamente. Isso salvará uma exceção para o app.
