Tutorial de Uso

1. Instalação

No Windows: Basta baixar o arquivo .exe e executá-lo. Não requer instalação formal.

No Linux: Após baixar o .AppImage, dê permissão de execução:

chmod +x RocknixManager.AppImage

2. Primeiro Acesso

  • Conecte o cartão SD do seu dispositivo Rocknix ao computador.
  • Abra o Rocknix Manager.
  • O programa deverá detectar automaticamente as partições de jogos (ROMs) e bios.

3. Funcionalidades Principais

Utilize a interface para organizar pastas, transferir arquivos de sistema e gerenciar backups de forma intuitiva, sem precisar lidar com partições Linux invisíveis no Windows padrão.
