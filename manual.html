Manual do Usuário

Aprenda a dominar todas as funções do Rocknix Manager.

1. Introdução

O Rocknix Manager foi criado para facilitar a gestão de cartões SD do sistema Rocknix. Como o sistema utiliza partições Linux (EXT4), muitos usuários de Windows têm dificuldade em acessar as pastas de jogos. Este programa resolve isso de forma simples e segura.

2. Preparação

  • Insira o cartão SD do seu console no computador usando um adaptador de qualidade.
  • Certifique-se de que nenhum outro programa de gestão de disco esteja aberto.
  • Execute o Rocknix Manager com privilégios de Administrador (necessário para ler partições).

3. Funções Principais

UI Gestão de ROMs

Acesse a aba de jogos para visualizar todas as pastas de consoles (SFC, MD, PS1, etc). Você pode arrastar e soltar arquivos diretamente para o cartão.

SYS Bios e System

Gerencie arquivos críticos do sistema e BIOS sem o risco de corromper a partição de boot. O programa protege arquivos vitais do Rocknix.

NPU Otimização de Performance

Configurações especiais para dispositivos que utilizam núcleos NPU para processamento acelerado de áudio e vídeo.

4. Solução de Problemas

Se o cartão não for detectado:

  • Verifique se o adaptador SD está bem conectado.
  • No Windows, abra o 'Gerenciamento de Disco' para ver se o cartão possui uma letra de unidade atribuída.
  • No Linux, certifique-se de usar sudo para rodar o AppImage.
